# Screen Sampler

A React + TypeScript app built with Vite, shadcn-ui, and Tailwind CSS.

## Development

Requires [bun](https://bun.sh/) to be installed.

```sh
# Install dependencies
bun install

# Start dev server
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview
```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
